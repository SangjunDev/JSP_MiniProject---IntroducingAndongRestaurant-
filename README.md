
# JSP_MiniProject_IntroducingAndongRestaurant

  주제 : 안동 소재지의 식당 소개 
  

# 프로젝트 기획서

  프로젝트명: AND Yummy

  프로젝트 개요

   - JSP를 활용하여 회원가입과 게시판 구현
   - 안동 식당들의 정보를 DB를 통해 관리
   - 동내별로 구분하여 식당 정보 제시

   
  프로젝트 주요 특징

    - 회원가입 및 게시판 구현
    - Daum 주소 API를 활용한 로그인 구현
    - DB를 활용하여 음식점 정보 관리
